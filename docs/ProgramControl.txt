Prompt:
responsible for connecting the command line to the file handler, requesting file contents
and returning readable text. When the program is executed with no arguments, this part of
the program should provide a list of files available, as provided from the work of
Team Member B, along with a corresponding number.
-------------
Class Structure:
ProgramControl.java
-Main class draws on methods from this
-This draws on methods from FileHandler, UserInterface
-Methods :
    requestFile(int index)
        Without index:
            listFiles()
            Query UserInterface to get the index
        Use the index to request the file contents from FileHandler
        Return file contents as String

    listFiles()
        Query FileHandler for files list
        Print out files list, displaying indices for each
-------------
Tests:
-requestFile:
    Cases:
        -empty args - returns listFiles()
        -1 args (valid) - return ciphered contents
            -invalid arg (<1, not an int, empty) - throw IllegalArgument Exception
        -2 args (valid) - return deciphered contents
            -first arg invalid (<1, not an int, empty) - throw IllegalArgumentException
            -second arg invalid (empty String) - return ciphered contents
        -more than 2 args - throw IllegalArgumentException
 -listFiles
    Cases:
        -call to fileHandler returns a String - return that String
        -call to fileHandler returns null - return null
